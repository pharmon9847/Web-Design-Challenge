Project Summary

The purpose of this project was to answer the question: "What's the weather like as we approach the equator?" To perform this analysis, data from 500+ randomly selected cities around the world were gathered using the OpenWeatherMap API. After the data was imported, I used the Pandas library to structure the data and Matplolib generate scatter plots:
  • Cloudiness Percentage vs. Latitude
  • Humidity Percentage vs. Latitude
  • Maximum Temperature vs Latitude
  • Wind Speed vs. Latitude
